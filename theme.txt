AARAMBH - DESIGN SYSTEM & ANALYSIS DOCUMENTATION
==============================================

1. BRAND ESSENCE
---------------
Purpose: Empowering aspiring entrepreneurs and innovators
Target Audience: Early-stage entrepreneurs, innovators, idea-stage ventures
Brand Personality: Professional, innovative, empowering, trustworthy
Core Message: "Transform Your Ideas Into Reality"

2. COLOR SYSTEM
--------------
Primary Colors:
- Deep Navy (#0A2647) - Represents trust and professionalism
- Coral Red (#FF6B6B) - Symbolizes energy and passion
- Success Green (#4CAF50) - Indicates growth and progress

Supporting Colors:
- Pure White (#FFFFFF) - Primary background
- Light Gray (#F5F5F5) - Secondary background
- Charcoal (#2C3E50) - Text and details

3. TYPOGRAPHY HIERARCHY
----------------------
Headings: Plus Jakarta Sans
- H1: 3.5rem / 700 weight - For hero headlines
- H2: 2.5rem / 600 weight - For section titles
- H3: 2rem / 600 weight - For subsections

Body Text: Inter
- Regular: 1rem / 400 weight
- Medium: 1rem / 500 weight
Line Height: 1.6 for optimal readability

4. LAYOUT STRUCTURE
------------------
Navigation:
- Minimal, fixed position
- Transparent background
- Key elements: Logo, Main Nav, Theme Toggle

Hero Section:
- Split-screen layout
- Left: Value proposition + CTA
- Right: Feature visualization
- Gradient accent background

Content Sections:
- Maximum width: 1440px
- Horizontal padding: 5vw
- Vertical spacing: 6rem between sections
- Card-based grid systems

5. COMPONENT DESIGN
------------------
Buttons:
Primary:
- Solid background
- Rounded corners (2rem)
- White text
- Scale animation on hover

Secondary:
- Transparent background
- Border outline
- Color matching text

Cards:
- Clean white background
- Soft shadow
- 1rem border radius
- Lift effect on hover
- Padding: 2rem

6. USER EXPERIENCE ELEMENTS
--------------------------
Micro-interactions:
- Button hover: Subtle scale (1.05)
- Card hover: Lift and shadow
- Links: Underline reveal
- Transitions: 0.3s ease

Animations:
- Page load: Fade-up entries
- Staggered animations (0.1s delay)
- Smooth scrolling
- Statistics counter animations

Key Sections:
1. Program Benefits

8. RESPONSIVE BEHAVIOR
---------------------
Breakpoints:
- Mobile: 320px
- Tablet: 768px
- Desktop: 1024px
- Wide: 1440px

Layout Adjustments:
Mobile:
- Single column
- Compact spacing
- Simplified animations

Tablet:
- Two columns where applicable
- Moderate spacing

Desktop:

- Comfortable spacing
- Enhanced interactions
